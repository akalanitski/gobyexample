Go па прыкладах: Strings and Runes

Тэкставы тым string у Go — гэта слайс байтаў, даступны толькі для чытання. Мова і стандартная бібліятэка апрацоўваюць тэкставыя зменныя асаблівым чынам — як кантэйнеры тэксту, закадзіраванага ў UTF-8. У іншых мовах радкі складаюцца з «сімвалаў». У Go паняцце сімвала называецца рунай. Руна – гэта цэлы лік, які прадстаўляе код Unicode. Гэты пост у блогу Go — добрае ўвядзенне ў тэму.

Каб падлічыць колькасць рун у радку, можна выкарыстоўваць пакет utf8. Звярніце ўвагу, што час выканання RuneCountInString залежыць ад памеру радка, таму што ён павінен дэкадаваць кожную руну UTF-8 паслядоўна. Некаторыя тайскія сімвалы прадстаўлены кодавымі кропкамі UTF-8, якія могуць ахопліваць некалькі байтаў, таму вынік гэтага падліку можа быць нечаканым.

 
    fmt.Println("Rune count:", utf8.RuneCountInString(s))

Цыкл range апрацоўвае тэкст дэкадуючы кожную руну разам з яе месцам.

    }

    for i, w := 0, 0; i < len(s); i += w {
        runeValue, width := utf8.DecodeRuneInString(s[i:])
        fmt.Printf("%#U starts at %d\n", runeValue, i)
        w = width
